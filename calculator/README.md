# Calculator

This is a simple Java console calculator that performs basic mathematical operations.

## Features

- Addition
- Subtraction
- Multiplication
- Division (with zero check)

## What I learned

- How to use Scanner for user input
- Basic arithmetic operations in Java
- Conditional statements (if/else)
- Error handling (division by zero)

## Technologies

- Java

## Purpose

The goal of this project is to practice basic programming logic and user input handling in Java.
