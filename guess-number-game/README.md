# Guess Number Game

This is a simple Java console game where the user tries to guess a randomly generated number.

## Features

- Random number generation
- User input with Scanner
- Loop until the correct number is guessed
- Hints (higher or lower)

## What I learned

- How to use loops (while)
- How to generate random numbers
- How to control program flow
- Basic game logic

## Technologies

- Java

## Purpose

The goal of this project is to practice programming logic and build a simple interactive console game.
